Installation

Prerequisites
  • Unity: Version 2020.3 or higher.
  • Git: Installed on your system.
  • Third-Party Assets: The following third-party assets are required but are not included in the repository. You must obtain and import them into Unity manually:
    • Massive Clouds - Screen Space Volumetric Clouds
    • Global Snow 2
    • DOTween
    • Text Animator for Unity
    • Third Person System 3
    • Dynamic Bone
    • FinalIK
    • Stylized Water 2
    • Dynamic Effects for Stylized Water 2 (Extension)
    • Underwater Rendering for Stylized Water 2 (Extension)
    • Pegasus

Steps to Get Started​

  1. Clone the repository: 
  git clone https://github.com/F1mmel/ZeldaTwilightPrincessRemastered.git

  1. Import the third-party assets listed above into the Unity project

  2. Download the ZeldaTPConfigurator

    The ZeldaTPConfigurator tool is essential for applying important modifications to your GameCube ISO before running the game. These adjustments ensure that the remastered version operates smoothly and provides the best possible experience.

    Open the Tool:
    Put the GameCube iso file into the "iso/" folder and launch the executable.

    Apply Modifications:
    Follow the on-screen instructions to:

    • Optimize the game's performance for modern systems.
    • Enable additional graphical enhancements and fixes.
    • Apply patches for compatibility with the remastered engine.

    Verify Integrity:
    The tool will perform a final check to ensure all required changes were applied successfully.

Important

The Configurator will not need delete or overwrite the iso file!

Problems?​

Feel free to ask for help on the Zelda Remastered Discord in the #tp-help channel!
